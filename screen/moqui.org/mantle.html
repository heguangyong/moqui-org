精选

从采购到支付

  • 管理供应商
  • 库存、设备、用品的采购订单
  • 接收和仓库管理
  • 应付账款
  • 从采购到支付测试中学习

从订单到收款

  • 支持和管理客户
  • 商品、订阅、服务的销售订单
  • 运费和税收计算（含Drools规则或集成）
  • 库存管理和履行
  • 应收账款和付款处理
  • 从订单到现金测试中学习

从退货到退款

  • 客户和供应商退货
  • 客户信用、退款、更换
  • 自动或手动授权和响应
  • 从退货响应测试中学习

从工作计划到费用

  • 客户请求
  • 项目、里程碑和任务
  • 灵活的费率、时间和费用
  • 账单客户、薪酬员工和分包商
  • 从工作计划到现金测试学习

产品信息

  • 商品、服务、订阅、设备类型
  • 使用类/类别/功能组织，刻面关键字搜索
  • 灵活的定价（并使用Drools规则进行修改）
  • 内容、识别、尺寸、协会、评论

资产管理

  • 库存、设备、用品等
  • GL发布发行、收据、折旧、实物库存
  • 基于预订、发货、生产运行等的库存预测
  • 设备调度和维护

人力资源和工资单

  • 管理职位、班级、员工
  • 灵活的小时费率和计件费率，工资
  • 工资税计算（联邦、州、地方；雇员和雇主份额），扣押
  • 通过支票、ACH等支付工资、税款、装饰

会计

  • 基于类型和类别的灵活帐户、报告和发布图表
  • 发票、付款、余额账户、资产、生产等的自动GL发布
  • 付款和GL对账（与OFX或其他），NACHA付款，正支付
  • 资产负债表、收入、现金流、权益表；财务比率和其他报告

通用

  • 多个组织、本地化和货币
  • 全系统搜索和分析
  • 访问和审计数据、可配置的授权和速度限制
  • 全面的REST API
